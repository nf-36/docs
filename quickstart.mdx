## Quickstart

Get your documentation site running and make your first edits.

### 1. Set up locally

1. Clone this repository.
2. Install the Mintlify CLI: `npm i -g mint`.
3. In your docs folder, run `mint dev` and visit `http://localhost:3000`.

### 2. Customize

Edit `docs.json` and the `*.mdx` files to add your own content and branding.

### 3. Publish

Commit and push your changes to deploy your updated docs.
